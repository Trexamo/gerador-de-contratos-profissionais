Política de Privacidade

Última atualização: 2024

1. Coleta de Dados

Coletamos os seguintes dados quando você usa nossa plataforma:

  • Dados de autenticação: Nome, email e foto do perfil (via Google Sign-In)
  • Dados de uso: Contratos gerados, histórico de atividades
  • Dados técnicos: IP, tipo de navegador, dispositivo

2. Uso dos Dados

Seus dados são usados para:

  • Fornecer e melhorar nossos serviços
  • Personalizar sua experiência
  • Comunicações sobre sua conta
  • Cumprir obrigações legais

3. Compartilhamento de Dados

Não vendemos seus dados. Podemos compartilhar com:

  • Processadores de pagamento: Mercado Pago (apenas dados necessários para transações)
  • Provedores de serviços: Hospedagem, suporte técnico
  • Autoridades legais: Quando exigido por lei

4. Seus Direitos (LGPD)

Você tem direito a:

  • Acessar seus dados pessoais
  • Corrigir dados incorretos
  • Excluir seus dados ("direito ao esquecimento")
  • Revoger consentimentos

5. Segurança

Implementamos medidas de segurança técnicas e organizacionais para proteger seus dados.
